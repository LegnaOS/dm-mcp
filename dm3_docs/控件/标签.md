# 标签

支持模板变量。

| 名称 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| 类型 | 文本 | true | 固定值： 标签 表示这是一个标签控件 |
| 名称 | 文本 | true | 窗口中唯一标识 |
| x | 整数 | false | 默认显示坐标x |
| y | 整数 | false | 默认显示坐标y |
| 宽度 | 整数 | false | |
| 高度 | 整数 | false | |
| 标题 | 文本 | false | |
| 字体 | 文本 | false | 资源标识 |
| 行距 | 整数 | false | |
| 间距 | 整数 | false | |
| 水平对齐 | 整数 | false | 0：左对齐 1：居中对齐 2：右对齐 |
| 垂直对齐 | 整数 | false | 0：顶对齐 1：垂直对齐 2：底对齐 |
| 自动换行 | 逻辑 | false | 标题超出限制宽度后是否换行显示 默认：false |
| 水平缩放 | 数值 | false | 仅艺术字模式下有效 |
| 垂直缩放 | 数值 | false | 仅艺术字模式下有效 |
| 文本色 | RGBA | false | |
| 描边色 | RGBA | false | |
| 背景色 | RGBA | false | |
| 边框宽度 | 整数 | false | 0：无边框 |
| 边框色 | RGBA | false | |
| 圆角半径 | 整数 | false | |
| 圆角边数 | 整数 | false | |
| 艺术字 | 文本 | false | 资源标识 |
| 数据源 | table | false | 设置一个kv table 模板标签会自动更新 |
| 默认可视 | 逻辑 | false | 默认是否显示 |
| 气泡提示 | 文本 | false | 需要开启 `鼠标事件` 才有效 |
| 气泡皮肤 | 文本 | false | |
| 鼠标事件 | 逻辑 | false | 是否响应鼠标事件,默认 `true` |
| 附加 | table | false | 自定义变量 |
| 系统事件 | table | false | |

```
{
 名称 = "调试标签",
 类型 = "标签",
 默认可视 = true,
 x = 14,
 y = 200,
 宽度 = 200,
 高度 = 30,
 水平对齐 = 1,
 垂直对齐 = 1,
 行距 = 2,
 间距 = 1,
 字体 = "宋体12",
 标题 = "你好啊",
 边框宽度 = 1,
 边框色 = RGBA(255,255,0,255),
 文本色 = RGBA(255,255,255,255),
 描边色 = RGBA(0,0,200,200),
 背景色 = RGBA(0,200,0,200),
 圆角半径 = 4,
 圆角边数 = 8,
 --艺术字
 --数据源
 系统事件 = {
 创建 = function(标签对象)
 调试输出(标签对象.名称)
 end
 }
},

```

## 属性

### 可视

- 类型：逻辑
- 说明：是否显示控件

### 父窗口

- 类型：窗口对象
- 读写：只读
- 说明：控件的父窗口对象

```
调试输出(标签1.父窗口.名称)

```

### 名称

- 类型：文本
- 读写：只读
- 说明：标签的名称。

### x

- 类型：整数
- 读写：读写
- 说明：标签的坐标x。

### y

- 类型：整数
- 读写：读写
- 说明：标签的坐标y。

### 宽度

- 类型：整数
- 读写：读写
- 说明：读写标签的宽度。

### 高度

- 类型：整数
- 读写：读写
- 说明：读写标签的高度。

### 次序

- 类型：数值
- 读写：读写
- 说明：动态改变控件的显示层级，值越大越先显示。

### 标题

- 类型：文本
- 读写：读写
- 说明：读写标题。

### 文本色

- 类型：RGBA
- 读写：读写
- 说明：读写标题文本色。

### 水平缩放

- 类型：数值
- 读写：读写
- 说明：水平缩放比例。

### 垂直缩放

- 类型：数值
- 读写：读写
- 说明：垂直缩放比例。

### 透明度

- 类型：小数
- 读写：读写
- 说明：取值范围 0-1

### 气泡提示

- 类型：文本
- 读写：读写
- 说明：读写控件的气泡提示。

### 数据源

- 类型：table
- 读写：读写
- 说明：取出/设置数据源。

### 附加

- 类型：table
- 读写：读写
- 说明：读写控件的附加信息。
