# 游戏盒子

## 介绍

- 接入平台的游戏可以在盒子中聚合展示。
- 可选配置文件位于盒子目录下的Dat\config.json。

| 名称 | 类型 | 说明 |
| --- | --- | --- |
| `title` | 文本 | 盒子的窗口标题 |
| `icon` | 文本 | 盒子的左上角图标文件,图标文件也位于 `Dat` 目录下 |
| `text` | 文本 | 盒子左下角状态栏中的信息 |
| `gid` | 整数 | 可以指定自己的工程ID作为首页。 |
| `uid` | 整数 | 作者账号uid,盒子内只显示指定作者游戏。 |

| | `{ "title": "游戏盒子", "icon": "窗口图标.png", "text": "一起玩游戏吧~", "gid": 100026 } ` |
| --- | --- |

```
{
	"title": "游戏盒子",
	"icon": "窗口图标.png",
	"text": "一起玩游戏吧~",
	"gid": 100026
}

```

提示

如果不填写gid盒子中将不会出现自定义的游戏首页。
