# 技能

## 介绍

技能对象的属性在脚本中可以随时随地修改。各个属性的说明参考技能配置项

## 属性

| 名称 | 权限 | 类型 | 说明 |
| --- | --- | --- | --- |
| 名称 | 只读 | 文本 | |
| 图标 | 只读 | 文本 | |
| 显示名 | 读写 | 文本 | |
| 气泡提示 | 读写 | 文本 | |
| 攻击距离 | 读写 | 整数 | |
| 最小距离 | 读写 | 整数 | |
| 间隔 | 读写 | 整数 | 毫秒 技能CD |
| 进度 | 读写 | 整数 | 范围：0-100 进度100时技能才可以释放 |
| hp | 读写 | 整数 | 需要消耗的hp |
| mp | 读写 | 整数 | 需要消耗的mp |
| 仇恨 | 读写 | 整数 | |
| 怒气 | 读写 | 整数 | |
| 等级 | 读写 | 整数 | |
| 伤害系数 | 读写 | 数值 | |
| 优先级 | 读写 | 整数 | 拥有多个自动释放的技能时，释放优先级越大越优先使用 -1 为禁用 |
| 自动释放 | 读写 | 逻辑 | |
| 格子效果 | 读写 | table | |

## 方法

### 刷新

刷新

说明：主动触发气泡提示变化。

示例 `技能对象.刷新() -- 会触发App.lua中[技能气泡提示]事件 `

```
技能对象.刷新() -- 会触发App.lua中[技能气泡提示]事件
```
