# 音乐

提示

音乐对象通过引擎.创建音乐创建。

## 属性

### 音乐

- 类型：文本
- 读写：读写
- 说明：取出/改变当前播放的音乐资源名称。

### 音量

- 类型：整数
- 读写：读写
- 说明：读写音量大小。

### 音高

- 类型：小数
- 读写：读写
- 说明：读写音高,越大声音越尖锐,默认：1.0。

### 声道

- 类型：小数
- 读写：读写
- 说明：范围：[-1, 1],表示左右声道,默认：0。

### 长度

- 类型：整数
- 读写：只读
- 说明：当前音乐的长度,单位：毫秒。

### 状态

- 类型：整数
- 读写：只读
- 说明：0：停止1：暂停2：播放。

### 循环

- 类型：逻辑
- 读写：读写
- 说明：当前音乐是否循环播放。

## 方法

### 播放

播放

说明：播放/继续播放 。

示例 `背景音乐.播放() `

## 示例

```
背景音乐.播放()

```

### 暂停

暂停

说明：暂停播放。

示例 `背景音乐.暂停() `

## 示例

```
背景音乐.暂停()

```

### 停止

停止

说明：停止播放。

示例 `背景音乐.停止() `

## 示例

```
背景音乐.停止()

```
