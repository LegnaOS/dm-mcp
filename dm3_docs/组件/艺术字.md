# 艺术字

## 配置项

| 名称 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| 类型 | 文本 | true | 固定值： 艺术字 表示这是一个艺术字组件 |
| 名称 | 文本 | true | 节点中唯一标识 |
| x | 整数 | true | 默认显示坐标x |
| y | 整数 | true | 默认显示坐标y |
| 宽度 | 整数 | true | |
| 高度 | 整数 | true | |
| 标题 | 文本 | true | |
| 艺术字 | 文本 | true | 资源标识,支持前缀 例："分组艺术字,10004_", |
| 间距 | 整数 | false | |
| 水平对齐 | 整数 | false | 0：左对齐 1：居中对齐 2：右对齐 |
| 垂直对齐 | 整数 | false | 0：顶对齐 1：垂直对齐 2：底对齐 |
| 文本色 | RGBA | false | |
| 默认可视 | 逻辑 | false | 默认是否显示 |
| 气泡提示 | 文本 | false | |
| 气泡皮肤 | 文本 | false | |
| 鼠标事件 | 逻辑 | false | 是否接收鼠标事件 |

```
{
 名称 = "血量标签",
 类型 = "艺术字",
 x = 0,
 y = 150,
 宽度 = 116,
 高度 = 20,
 间距 = -2,
 水平对齐 = 2,
 垂直对齐 = 1,
 艺术字 = "分组艺术字,10004_",
 标题 = "6",
 文本色 = RGBA(255,255,255,255),
},

```

## 属性

### 可视

- 类型：逻辑
- 说明：是否显示组件

### 名称

- 类型：文本
- 读写：只读
- 说明：组件的名称。

### 类型

- 类型：文本
- 读写：只读
- 说明：组件的类型。

### x

- 类型：整数
- 读写：读写
- 说明：组件的x坐标。

### y

- 类型：整数
- 读写：读写
- 说明：组件的y坐标。

### 标题

- 类型：文本
- 读写：读写
- 说明：读写标题。

### 文本色

- 类型：RGBA
- 读写：读写
- 说明：读写文本色。

### 气泡提示

- 类型：文本
- 读写：读写
- 说明：读写气泡提示。

### 鼠标焦点

- 类型：逻辑
- 读写：只读
- 说明：组件当前是否有鼠标焦点。

### 鼠标坐标

- 类型：table {x,y}
- 读写：只读
- 说明：组件当前的鼠标坐标。

### 次序

- 类型：数值
- 读写：读写
- 说明：动态改变节点的显示顺序，值越大越先显示。

## 方法

### 销毁

销毁

说明：销毁当前组件。

示例 `本次焦点.销毁() `

## 示例

```
本次焦点.销毁()

```
