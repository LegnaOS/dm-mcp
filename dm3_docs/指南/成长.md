# 成长

## 介绍

自动为角色配置对应等级的基础属性。

```
local 成长配置 =
{
	类型 = "成长",
	名称 = "主角成长",
	数据 = {
		{maxhp=100,maxmp=100,....}, -- 1 级
		... -- 2 级
		.. -- 3 级
		... -- 4 级
	}

}

return 成长配置

```

```
local 成长表 = {}

for n=1,99 do
	local 成长信息 = {
		maxhp = 60 + (n-1) * 5,
		maxmp = 30 + (n-1) * 3,
		防御 = 0,
		攻击 = 5,
		魔法 = 0,
		道术 = 0,
		经验值 = 0,
		升级经验 = math.floor((50 * (n ^ 1.8 + 0.4 * n)) / 10 + 0.5) * 10
	}

	table.insert(成长表,成长信息)
end

local 成长配置 =
{
	类型 = "成长",
	名称 = "主角成长",
	数据 = 成长表
}

return 成长配置

```

## 配置项

| 名称 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| 类型 | 文本 | true | 固定值： 成长 表示这是一个成长组件 |
| 名称 | 文本 | true | 全局唯一的成长标识 |
| 数据 | table {} | true | 等级对应成员索引 |
