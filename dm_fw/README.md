# DM3 框架模板

这是一个预配置的 DM3 游戏项目模板，包含了基础的目录结构和配置文件。

## 目录结构

```
dm_fw/
├── App.lua              # 游戏主配置
├── Server.lua           # 服务器配置（可选）
├── 角色/                # 角色配置目录
├── 道具/                # 道具配置目录
├── 技能/                # 技能配置目录
├── 状态/                # 状态配置目录
├── 窗口/                # 窗口配置目录
├── 地图/                # 地图配置目录
├── 资源/                # 资源文件目录
│   ├── 图片/
│   ├── 音效/
│   └── 音乐/
└── 脚本/                # 自定义脚本目录
```

## 使用说明

创建新项目时，MCP 服务器会：
1. 复制整个 dm_fw 目录到项目位置
2. 重命名为项目名称
3. 自动配置 App.lua
4. 创建必要的子目录

## 模板特点

- ✅ 完整的目录结构
- ✅ 预配置的 App.lua
- ✅ 示例配置文件
